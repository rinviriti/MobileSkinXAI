# 🩺 MobileSkinXAI

An Explainable Cross-Domain Lightweight Deep Learning Framework for Mobile-Based Skin Cancer Screening.

## Overview

MobileSkinXAI is a research-driven project that aims to develop a lightweight and explainable AI system for skin cancer screening on mobile devices.

The framework focuses on four major skin lesion categories:

- Benign Nevus
- Melanoma
- Basal Cell Carcinoma (BCC)
- Squamous Cell Carcinoma (SCC)

The project combines lightweight deep learning architectures with explainable AI techniques to provide interpretable predictions suitable for mobile healthcare applications.

---

## Research Objectives

- Develop a lightweight skin lesion classifier
- Improve cross-domain generalization
- Generate explainable predictions using Grad-CAM
- Deploy the model in a Flutter mobile application

---

## Dataset

### ISIC Dataset
- Dermoscopic skin lesion images

### PAD-UFES-20 Dataset
- Clinical skin lesion images
